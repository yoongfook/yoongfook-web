Useful list of Ruby gems that I frequently use in web development projects.

Front End Frameworks

  • Zurb Foundation - Uses SASS
  • Twitter Bootstrap - Uses LESS

Markup/Styling

  • HAML - Prettify your HTML. Looks much cleaner than erb.
  • SASS - Avoid spaghetti CSS. Uses SASS today.
  • Font-Awesome - Icon-fonts

Forms

  • Simple Form - Form generator: labels, error states, dropdowns, checkboxes, etc

Error Reporting

  • Better Errors - Meaningful error screens
  • Rails Panel - Show Rails app requests in Chrome Dev Tool

Authentication

  • Devise - THE authentication system for Rails

Authorization

  • Omniauth - Login with Facebook, Twitter, g+ etc.
  • CanCan - Role-based Authorization (RBA) system that restricts what resources a given user is allowed to access

File/Image Upload

  • CarrierWave - Upload images/files from Rails application

Asset Pipeline

  • Fog - Upload images to cloud
  • Asset Sync - Sync and serve assets from S3, Cloudfront etc.

Pagination

  • Kaminari - Easily create pagination when entries from db queries get too long.

Background Tasks

  • Delayed Job - Execute longer tasks like emailing a long list of entries in the background.

Admin Panels

  • Active Admin - Instantly create an admin panel for your rails app.
  • Rails Admin - Another great admin panel for managing your data

Content Management System

  • RefineryCMS - Popular Rails based CMS
